# ⟁ Activator Demo GIF

**Visual demonstration of the MirrorDNA activation process**

---

## GIF Specifications

**Duration**: 10-15 seconds
**Format**: Animated GIF or MP4
**Resolution**: 1200x800px minimum
**Frame Rate**: 15-30 fps
**File Size**: <5MB (for web performance)

---

## Storyboard

### Frame 1 (0-2s): Opening Screen
- Show a blank ChatGPT or Claude chat interface
- Title overlay: "Activating MirrorDNA in 15 seconds"

### Frame 2 (2-4s): Copy Activator
- Highlight the `MirrorDNA-Activator.txt` file in a file browser
- Cursor selects all text (Ctrl+A)
- Copy animation (Ctrl+C)
- Visual cue: Clipboard icon or checkmark

### Frame 3 (4-6s): Paste into Chat
- Switch to AI chat interface
- Cursor moves to message box
- Paste text (Ctrl+V)
- Show first few lines of activator appearing:
  ```
  ⟁ MIRRORDNA ACTIVATOR ⟁
  ═══════════════════════
  ```

### Frame 4 (6-8s): Send & Activation
- Click "Send" or hit Enter
- Brief loading animation
- AI responds with:
  ```
  ⟁ INITIALIZATION COMPLETE

  MirrorDNA™ protocol is now active.
  ```

### Frame 5 (8-10s): Quick Test
- User types: "I'm working on a web app. I prefer simple solutions."
- AI responds with:
  ```
  ⚡ Got it—I'll keep things practical and straightforward.
  What's your first challenge?
  ```
- Highlight the word "practical" matching "simple" from user

### Frame 6 (10-12s): Continuity Demo
- User types: "What database should I use?"
- AI responds referencing earlier context:
  ```
  ∴ Given you prefer simple solutions, start with SQLite...
  ```
- Highlight "simple solutions" being remembered
- Checkmark or sparkle animation around the reference

### Frame 7 (12-15s): End Card
- Fade to final frame
- Text: "⟁ MirrorDNA: Paste. Activate. Reflect."
- Button/link: "Try it now → github.com/MirrorDNA..."

## Visual Style

**Color Palette**:
- Background: Dark mode (matching Claude/ChatGPT dark theme)
- Accent: Purple/blue gradient for highlights
- Glyph colors: Gold/amber for ⟁ and special symbols

**Typography**:
- Monospace font for code/activator text
- Sans-serif for UI elements
- Large, readable text (16px minimum)

**Animations**:
- Smooth typing effect (not instant paste)
- Gentle fade transitions between frames
- Highlight pulses for key moments (continuity callbacks)
